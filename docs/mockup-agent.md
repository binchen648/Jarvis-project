## 三大改动

### 改动 1: 双栏布局 → Agent = 控制台

| 左侧：Conversation | 右侧：Active State |
|-------------------|-------------------|
| 用户 ↔ Agent 的对话流 | 当前目标 (进度条 + 截止日) |
| 工具调用卡片 | 当前记忆 (自动加载相关笔记) |
| AI 建议 + 执行按钮 | 今日计划 (同步自 Dashboard) |
| 纯消息历史 | 最近工具调用记录 |

右侧 Active State **始终可见**，不随对话滚动。用户随时知道 Agent 当前在关注什么。

---

### 改动 2: 建议 → 执行闭环

Agent 给出建议后，每条建议右侧跟着操作按钮：

```
agent: 我建议：

1. 今晚完成 LeetCode 92 (35min)
   [开始]  [加入计划]  [忽略]

2. 复习哈希表笔记 (10min)
   [开始]  [加入计划]  [忽略]
```

| 按钮 | 行为 |
|------|------|
| **开始** | 直接跳转到目标页面，进入执行模式 |
| **加入计划** | 添加到 Today's Plan（同步到 Dashboard） |
| **忽略** | 从建议列表移除 |
| 全部接受 | 一键将所有建议加入计划 |

**关键设计**: 建议不终止于建议，而是形成 **建议 → 行动 → 完成** 的闭环。

---

### 改动 3: AI Surface 在 Agent 页面收缩

```
离开 Agent 页面:
    ✦✦  ← 完整的 AI Surface + Pulse + 卡片
    右下角 24px

在 Agent 页面:
    ✦   ← 仅保留 Pulse 呼吸动画
    右下角 24px
    无 Surface 卡片，无弹出内容
    避免"两个 AI"的视觉重复

Agent Thinking 时:
    ✦✦✦  ← Pulse 加速 (快速紫色脉冲)
    与 Active State 中的工具调用同步
```
