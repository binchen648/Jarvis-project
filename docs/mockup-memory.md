## 设计理念

### 记忆 ≠ 操作日志

| 操作日志 | Jarvis Memory |
|---------|--------------|
| 时间戳 + 事件 | 时间戳 + 事件 + **AI 理解** |
| 无结构扁平列表 | **按类型分类** + 彩色标识 |
| 不可交互 | **可点击跳转** + 关联记忆 |
| 无 AI 注解 | **AI 自动补充**相关度/建议 |
| 纯文本 | **Rich card** 含进度/时长/专注度 |

每条记忆卡片不仅仅是"记录"，还包含 AI 对这条记忆的理解和相关建议。

## AI Surface 在 Memory 页面的行为

```
进入 Memory 页面:
    AI Surface 完全恢复 (不同于 Agent 页面的收缩)
    ✦✦     ← Pulse + Surface 卡片

点击/悬停某条记忆时:
    Pulse 切换为青色涟漪 (Memory Pulse)
    Surface 卡片显示对应记忆的 AI 注解

搜索记忆时:
    Pulse 变为搜索模式 (快速闪烁)
    Surface 卡片显示 "正在搜索..."
```
